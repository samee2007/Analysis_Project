# 📊 Sales Analysis

## 📌 Objective
Analyze sales data to understand trends, top-performing products, and regional performance.

## 🛠 Tools Used
- Python
- Pandas
- NumPy
- Matplotlib

## 📊 Analysis Performed
- Total Sales Calculation
- Region-wise Sales Analysis
- Top Selling Products
- Monthly Sales Trends

## 📈 Key Insights
- Some regions generate higher revenue than others
- A few products contribute most of the sales
- Sales show variation across months

## 📁 Files Included
- Dataset (CSV file)
- Jupyter Notebook (`analysis.ipynb`)
