# 🛒 E-Commerce Analysis

## 📌 Objective
Analyze online retail data to understand customer behavior and revenue patterns.

## 🛠 Tools Used
- Python
- Pandas
- NumPy
- Matplotlib / Seaborn

## 📊 Analysis Performed
- Total Revenue Calculation
- Revenue by Country
- Top Selling Products
- Top Customers
- Monthly Revenue Trends

## 📈 Key Insights
- Majority of revenue comes from specific countries
- A small number of customers generate high revenue
- Certain products dominate sales volume

## 📁 Files Included
- Jupyter Notebook (`analysis.ipynb`)
