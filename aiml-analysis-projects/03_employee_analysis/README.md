# 👨‍💼 Employee Attrition Analysis

## 📌 Objective
Analyze employee data to identify factors affecting attrition.

## 🛠 Tools Used
- Python
- Pandas
- NumPy
- Matplotlib / Seaborn

## 📊 Analysis Performed
- Overall Attrition Rate
- Department-wise Attrition
- Age Group Analysis
- Salary vs Attrition

## 📈 Key Insights
- Younger employees have higher attrition
- Some departments show higher turnover
- Income level may influence attrition

## 📁 Files Included
- Dataset (CSV file)
- Jupyter Notebook (`analysis.ipynb`)
